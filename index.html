Projects

Remote Robot Manipulation Anthropomorphic

We developed an advanced remote control system for an anthropomorphic arm to improve human-robot interaction in industrial environments. This project carried out in collaboration with my colleague Agustín Sardi Bottino, addressed the need to provide more intuitive and natural control of robotic manipulators.

The product combines computer vision and remote control technologies to allow operators to manipulate the robotic arm precisely and safely. We implemented a gesture capture system based on computer vision that translates the operator's movements into commands for the robotic arm. This innovative approach enhances the precision and ease of use of robotic control.

We developed robust control algorithms using Python and C++ programming to program the Blue Pill board, ensuring efficient and reliable operation of the robotic arm. Additionally, we created interactive training software using Unity, which simulates the work environment and facilitates the learning and practice of operators, allowing them to familiarize themselves with the system before using it in a real setting.

The anthropomorphic arm we assembled features high-quality mechanical and electronic components designed to withstand the demands of industrial operations. The final prototype integrated all the necessary hardware with the control software, achieving a cohesive and functional solution.

This remote control system represents a significant advancement in human-robot interaction, offering a powerful tool to improve efficiency and safety in various industrial applications. The combination of computer vision, advanced programming, and Unity simulation provides intuitive and precise handling, facilitating the work of operators and enhancing productivity in the industrial environment.

3D Printer

My 3D printer project is based on the renowned Prusa 3D model and operates using the open-source Marlin firmware. This project incorporates several key technical specifications and components for optimal performance. The printer's frame is constructed from MDF (medium- density fiberboard)

At the core of the control system is an Arduino Mega, paired with a RAMPS 1.6 kit. The Arduino Mega, known for its capacity and flexibility, runs the Marlin firmware, which is highly popular in the 3D printing community for its reliability and customization options. The RAMPS 1.6 kit facilitates the connection and control of stepper motors, heaters, and temperature sensors, enabling efficient and precise management of all the printer's components. This configuration ensures the printer can handle complex tasks and produce high -quality prints.

The printer is designed to offer a 20 cm by 20 cm print area, which is suitable for a wide range of projects, from small prototypes to larger, more detailed pieces. This generous print space, combined with the sturdy MDF frame and advanced control system based on the Arduino Mega and RAMPS 1.6, makes this printer a powerful and versatile tool for any 3D printing enthusiast. With these features, the printer is not only capable of producing precise and detailed prints but also serves as an excellent platform for experimentation and learning in the field of 3D printing.
